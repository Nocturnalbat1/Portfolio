Projects

Here are a few projects I completed.

AITHREATDETECT – Edge AI Threat Detection

Onboard threat detection for spacecraft-style bus traffic running on a Snapdragon RB5. Built a prototype architecture to ingest telemetry/command traffic, analyze it with an AI model, and output JSON-based threat reports under tight resource constraints.

Tools: Python · Qualcomm RB5 · F´ Framework

Xworm Malware Dynamic Analysis

Sandboxed execution and behavior analysis of Xworm with ANY.Run and Cuckoo Sandbox. Documented persistence mechanisms, network C2 traffic, and mapped IOCs to MITRE ATT&CK, then proposed blue-team detections.

Tools: ANY.Run · Cuckoo Sandbox · Wireshark · MITRE ATT&CK

Python: Rock Paper Scissors

Small game focused on input handling, control flow, and conditional logic. Used this project to tighten fundamentals before moving into more complex automation and security-focused scripting.

Tools: Python 3.11 · Numpy · Random · time

Risk Management Simulation

Turn-based Rock Paper Scissors–style simulation to practice decision-making, probabilities, and risk trade-offs. Emphasizes reading outcomes and adapting strategy in later rounds.

Tools: Python 3.11 · Numpy · Random · time

HTB “Meow” CTF

Walkthrough of HackTheBox “Meow” machine. Focused on enumeration, basic service abuse, and building repeatable note-taking for future CTF and lab work.

Tools: Nmap · Netcat · Linux CLI

SQL Database
