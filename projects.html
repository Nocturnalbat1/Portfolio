Projects

Here are a few of the projects I’m most proud of.

AITHREATDETECT – Edge AI Threat Detection

Onboard threat detection for spacecraft-style bus traffic on Snapdragon RB5.

Built a prototype architecture to ingest telemetry/command traffic, analyze it with an AI model running on the RB5, and output JSON-based threat reports. Focused on constraints like limited compute, near real-time monitoring, and simple CLI workflows.

Tools: Python · Qualcomm RB5 · F´ · JSON Schema · Linux

Xworm Malware Dynamic Analysis

Sandboxed execution, IOCs, and MITRE ATT&CK mapping.

Used environments like ANY.Run and Cuckoo Sandbox to monitor Xworm’s behavior, including process creation, persistence mechanisms, registry changes, and network C2 traffic. Documented IOCs and mapped them to ATT&CK techniques, then proposed blue-team responses.

Tools: ANY.Run · Cuckoo Sandbox · Wireshark · MITRE ATT&CK

Python: Rock Paper Scissors

Building a small Rock Paper Scissors game for users.

Built the game loop, handled user input, and implemented conditional decision-making to determine win/loss outcomes. This project helped me strengthen my fundamentals in control flow, input validation, and basic game logic solidifying my foundation before moving into more complex automation and security-focused scripts.

Tools: Python3.11 · Numpy · Random · time
